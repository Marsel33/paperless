Регистратор штрихкодов

Регистратор штрихкодов - это одно из веб-приложений подсистемы безбумажного производства с открытым исходным кодом, разработанное компанией Окнософт на базе фреймворка Metadata.js.
Исходный код и документация доступны на GitHub .

- Для промышленной эксплуатации рекомендуется google chrome.
- Поддержаны все современные мобильные браузеры.
- Возможна компиляция веб-приложения в формат нативных мобильных или десктопных приложений.

Ключевые возможности и особенности

- Регистрация штрихкода, полученного сканерами usb, bluetooth, водом с клавиатуры или камерой мобильного устройства.
- Полная интеграция с 1С. Возможность показать любой отчет, диаграмму или иные данные ИБ 1С с поддержкой ссылочной типизации.
"Из коробки" поддержано взаимодействие с платформой >= 8.3.6. При необходимости, можно реализовать шлюз в 1С версий 7.7 или 8.2.
- Поддержка автономной работы. Приложение может работать при выключенном сервере, перебоях или полном отсутствии связи. Результаты сканирования сохраняются в базе данных браузера и отправляются на сервер, как только он становится доступен.
- Визуальная, звуковая и вибро обратная связь при сканировании. Система может контролировать корректность штрихкодов и информировать оператора о результатах сканирования.

 

Использованы следующие библиотеки и инструменты:

Серверная часть

  • 1c_enterprise, ORM сервер 1С:Предприятие
  • postgreSQL, мощная объектно-раляционная база данных
  • node.js, серверная программная платформа, основанная на движке V8 javascript
  • nginx, высокопроизводительный HTTP-сервер

Управление данными в памяти браузера

  • alaSQL, база данных SQL для браузера и Node.js с поддержкой как традиционных реляционных таблиц, так и вложенных JSON данных (NoSQL)
  • xmlToJSON, компактный javascript модуль для преобразования XML в JSON
  • xlsx, библиотека для чтения и записи XLSX / XLSM / XLSB / XLS / ODS в браузере

UI библиотеки и компоненты интерфейса

  • dhtmlx, кроссбраузерная библиотека javascript для построения современных веб и мобильных приложений
  • filesaver.js, HTML5 реализация метода saveAs

Графика

  • fontawesome, набор иконок и стилей CSS
